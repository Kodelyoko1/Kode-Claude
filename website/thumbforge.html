FAQ

Common questions

What image format does ThumbForge output?
PNG files at the exact YouTube recommended dimensions: 1280×720 for the standard thumbnail and 1080×1920 for the Shorts variant. Upload them directly — no resizing needed.
Can I use my own font?
The default renders use system-available fonts (bold sans-serif). Drop your own TTF/OTF font file into the agent's fonts directory and reference it by filename in the manifest JSON. The agent uses it automatically for that slug.
How do I add a face or graphic to the thumbnail?
Add an "overlay" path to your manifest pointing to a transparent PNG cutout. ThumbForge composites it onto the thumbnail at the right edge — the standard "face + bold text" layout that dominates high-CTR thumbnails.
Does it work with the CarouselForge or ShowNotes agents?
Yes — if you've already run ShowNotes, the episode title and subtitle are in the show notes markdown. Copy them into a ThumbForge manifest to generate a matching thumbnail for the companion YouTube video in under 30 seconds.
