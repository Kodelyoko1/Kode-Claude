How it works

One platform, many revenue streams.

Each agent is a standalone Python script with a paywall gate, Gmail-SMTP email output, and flat JSON data storage. Run one or run all of them.

1

Set your env vars

Drop SMTP_USER, SMTP_PASS, and AGENT_PASSWORD in .env. Most agents need nothing else.

2

Run the agent

Every agent is python3 run_<agent>_auto.py. Wire it into cron or run_all_autonomous_agents.sh.

3

Collect the output

Leads, reports, newsletters, audio files, carousel images — all land in data/ and get emailed to you automatically.

4

Gate external clients

Subscribers pay via PayPal or Stripe. The paywall system manages access keys, subscription records, and renewal emails without you lifting a finger.
