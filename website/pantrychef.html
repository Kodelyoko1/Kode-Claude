3 Steps

Drop a list. Get a week.

No login, no recipe database to browse, no subscription to a meal kit service.

List your pantry

Create a data/pc_users/you.json file with your ingredients, serving size, and any foods to avoid. The format is a simple JSON array — no nutrition IDs or database lookups.

Run PantryChef

Execute python3 run_pantrychef_auto.py. The agent reads your pantry, selects 7 pantry-optimized meals, and calculates the minimum grocery delta to complete the week.

Get your plan

A markdown file lands in data/pc_plans/ with the 7-day meal table, prep times, and the grocery list. Copy to Notion, print it, or text it to whoever's cooking.

FAQ

Common questions

How do I tell PantryChef what's in my pantry?
Drop a data/pc_users/your-name.json file with a pantry array of ingredient strings: {"name": "Alex", "pantry": ["chicken breast", "pasta", "canned tomatoes"], "servings": 2, "avoid": ["nuts"]}. That's the whole setup — no database IDs, no barcodes.
Does it handle dietary restrictions?
Yes — add an "avoid" list to your user file with any ingredients or categories to skip (dairy, gluten, nuts, pork, shellfish, etc.). The planner skips any recipe that relies on those ingredients.
What does the output look like?
A markdown file in data/pc_plans/ with a 7-row table: day, meal name, cook time, pantry ingredients used, and items to buy. Paste it into Notion, a Notes app, or print it and stick it on the fridge.
Can I use it for meal prep rather than weeknight cooking?
Yes. The 30-Day Package runs a 4-week plan at once. Add "meal_prep": true to your user file and the planner front-loads batch-friendly recipes on Sundays — soups, grain bowls, and casseroles that reheat well through the week.
