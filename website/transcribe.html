FAQ

Common questions

How accurate is the transcription?
faster-whisper's "base" model gets 85–92% word accuracy on clear speech with minimal background noise. For interviews with two people talking cleanly, expect 90%+. Heavy accents or noisy recordings may dip lower — a light proofread pass fixes most errors quickly.
How long does it take to transcribe an hour of audio?
On a modern CPU (no GPU), faster-whisper processes roughly 1 hour of audio in 20–35 minutes using the int8 quantized model. Batch runs overnight so you wake up to finished files.
Do I need a GPU or cloud account?
No. The agent runs on CPU using the int8-quantized model — no GPU, no CUDA, no cloud API. The only dependencies are Python, faster-whisper, and ffmpeg, all installable with pip and apt/brew.
Can I use the transcripts with the ShowNotes agent?
Yes — that's exactly what they're designed for. Transcribe writes to data/tr_outputs/ and ShowNotes auto-ingests from that same folder. Run Transcribe first, then run ShowNotes to get a full show-notes package from your episode in two commands.
