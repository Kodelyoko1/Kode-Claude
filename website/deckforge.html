Questions

Can I edit the slides after delivery?＋

Yes. The output is a standard HTML file — you can open it in any text editor and change text, colors, or structure. The Reveal.js library is loaded from CDN so no Node.js or build step is needed. If HTML editing isn't your thing, we include a design brief that maps each slide's content so you can hand it to a designer.

Does the PDF export look professional?＋

Yes, when you use Chrome's "Save as PDF" print dialog at 100% scale with background graphics enabled. Chrome renders CSS pixel-perfect to PDF. We test every output before delivery. If you want a vector-native PowerPoint instead, request that at checkout and we'll quote a conversion.

What if I don't have traction numbers yet?＋

Pre-revenue founders still need decks. Just tell us what you have: beta waitlist size, pilot customers, LOIs signed, user interviews conducted, or market research. We'll frame whatever proof of demand you have as credibly as possible on the traction slide.

What information do you need from me?＋

After paying, email us: (1) company name + tagline, (2) the problem you solve in 2 sentences, (3) your solution in 2 sentences, (4) any traction metrics, (5) team bios (1 line each), (6) raise amount and use of funds, (7) preferred theme (dark/light/minimal), (8) optional brand hex color.
