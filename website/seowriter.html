Questions

Do I need a Claude API key?＋

No. The heuristic skeleton runs without any API keys and produces a complete article structure at zero per-article cost. Add ANTHROPIC_API_KEY to your environment to unlock full prose drafts — each section becomes a fully written paragraph rather than an outlined placeholder.

How long are the articles?＋

Heuristic outlines run 800–1,200 words. Claude-powered full drafts reach 1,500–2,500 words depending on the niche, keyword density, and number of H2 sections generated. FAQ sections add another 300–500 words on top.

What file format does it output?＋

Markdown (.md) files saved to data/sw_outputs/. Drop them straight into Ghost, Hashnode, or paste into WordPress's block editor. If you use Substack, paste the raw text — it imports cleanly. Pair with the Localize agent to translate before publishing.

Can it auto-publish?＋

Not yet — output is draft-ready markdown. Auto-publish to WordPress or Ghost requires a CMS API integration that isn't currently wired in. Use your CMS's import function or the Markdown copy-paste flow. Pair with the Localize agent to translate first before publishing to multilingual audiences.
