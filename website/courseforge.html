How It Works

Frequently Asked Questions

Do I need to have content from other agents to use CourseForge?

No — any markdown or text files work as source material. However, CourseForge is designed to auto-detect outputs from ShowNotes, SEOWriter, Transcribe, PaperBrief, GutenbergVoice, and StoryForge for the smoothest workflow.

What platforms does the bundle work with?

Gumroad, Payhip, Udemy, Teachable, and any platform that accepts a ZIP of files. Specify your platform in the manifest and CourseForge adjusts module formatting and landing page structure accordingly.

Can I create a course with video lessons?

CourseForge produces text-based course bundles (markdown modules). For courses with video, pair it with Transcribe to get transcripts, then CourseForge wraps those into structured lesson modules your students can read alongside the video.

What's in the manifest.json output file?

A structured JSON file with course metadata — title, description, price, modules list with estimated reading times, target platform, and a list of all included files. Useful if you want to build your own custom course delivery system later.
