FAQ

Common questions

Do I need a designer to use the Canva brief?
No. The spec sheet gives you every value you need to click-and-place in Canva's free editor. Background color, font name, size, weight, element position — everything is specified. Most people finish the rebuild in under 10 minutes.
What format is the manifest?
A JSON file: {"slug": "ig-post-june", "format": "ig_post", "headline": "Your text", "handle": "@yourhandle", "accent": "#833ab4", "bg": "gradient"}. Drop it in the inputs folder and run the agent.
Can I order the same template in multiple formats?
Yes — create one manifest per format slug. The 3/month plan covers three separate format renders, so ordering one design across IG post + IG Story + LinkedIn uses all three slots for that month.
