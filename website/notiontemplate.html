FAQ

Common questions

Do I need a Notion API key?
No. The agent generates all four output files without any Notion credentials. The template_spec.md gives you step-by-step instructions to build it manually in under 10 minutes. The Notion API push is an optional bonus — set NOTION_API_KEY and NOTION_PARENT_PAGE_ID in your .env and the template appears in your workspace automatically.
Can I request a preset that isn't listed?
Yes — on the Unlimited plan, custom preset requests are included. Describe the database structure you need (what you're tracking, which properties matter) and the agent will design and generate a new preset schema for you. Custom presets are also available as a one-off add-on on the Single plan for an additional $19.
What format is the template.json in?
The template.json follows the Notion API database schema shape — the same structure used by POST /v1/databases. If you have a Notion integration token, you can push it directly via the API or use a tool like Postman. If not, the template_spec.md translates every property into plain-English Notion UI steps.
Can I resell the templates?
Yes. Once you purchase a template bundle you have full rights to publish and sell it on Gumroad, Payhip, Lemon Squeezy, or your own site. The included landing_page.md is already written as a sales page so you can go live in minutes. Attribution is not required.
