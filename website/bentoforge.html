FAQ

Common questions

Where do I host the HTML file?
Anywhere that serves static HTML — GitHub Pages, Netlify, Cloudflare Pages, and Vercel all have free tiers that deploy in under a minute. Point your Instagram bio link to that URL and you're live.
Can I update the page later?
Yes. Edit your manifest JSON and re-run the agent to get a new HTML file. Re-deploy it to your host (usually a git push or drag-and-drop). The URL stays the same.
Does it track clicks?
The default output has no tracking — clean HTML, no scripts. Add a "analytics_id" key to the manifest to inject a Google Analytics or Plausible snippet into the output automatically.
