Questions

How do I add content to my niche?＋

Save your niche source pages as HTML files (File → Save Page As in your browser) and drop them into the agent's snapshot folder. NicheLens reads, parses, and ranks the stories automatically each week. No API access to external sites is required — you control what goes in.

What affiliate networks work with NicheLens?＋

Any network that gives you a standard affiliate URL — Amazon Associates, ShareASale, Impact, PartnerStack, or direct-brand affiliate programs. You provide your tracked links in the niche config file; the agent matches them to relevant stories based on keyword rules you define.

Can I run multiple niches?＋

Yes — each niche is a separate subscription at $7/mo (or $59/yr). Add as many as you like. Each one gets its own digest cadence, affiliate link config, and subscriber list. There is no volume cap on niches.

How are affiliate links matched to content?＋

You configure a keyword-to-affiliate-URL map in the niche config file. When a story's title or summary contains a matched keyword (e.g., "Orion telescope"), the corresponding affiliate link is appended to that story row in the digest. Multiple affiliate matches in one issue are supported — each annotated with an [affiliate] label for FTC compliance.
