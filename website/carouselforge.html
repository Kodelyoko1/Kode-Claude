FAQ

Common questions

How many slides does each carousel get?
As many as your slides array has entries, plus the auto-generated cover and CTA. Most carousels run 6–10 total slides. LinkedIn's algorithm tends to favor 7–10 slides; Instagram performs well at 5–8. The CTA slide is always the last one.
How does ShowNotes auto-ingest work?
When you run the ShowNotes agent it writes a markdown file to data/sn_outputs/ that includes a "Key Takeaways" section. CarouselForge reads that section and converts each bullet into a slide body. You still set the platform and theme via a lightweight config file — the content fills itself in.
Can I add images to individual slides?
Yes — add an "image" path to any slide entry in your manifest. CarouselForge composites it into the slide at a 40% opacity background wash behind the text, keeping readability while adding visual interest.
What files do I actually upload to Instagram or LinkedIn?
Each slide is a separate PNG in data/cr_outputs/{slug}/, numbered sequentially (01-cover.png, 02.png, ..., 06-cta.png). Upload them in order as a native multi-image post on Instagram or a Document post on LinkedIn.
