Questions

What transcript formats are supported?＋

Plain .txt and .srt (SubRip) files. Drop them into data/sn_inputs/ or let them auto-arrive from the Transcribe agent's output folder. If you have an audio file first, run it through Transcribe — it outputs both a .txt and a .srt that ShowNotes picks up automatically.

Do chapter timestamps require an SRT file?＋

Yes — exact timestamps come from SRT timecodes. For plain .txt transcripts without timing data, the agent still outputs a full TL;DR, key takeaways, and resource links; chapter timestamps show as [estimated] placeholders instead of precise times.

Do I need a Claude API key?＋

No. The heuristic engine extracts takeaways, resource links, and SEO meta without any API key. Add ANTHROPIC_API_KEY to your environment if you want a polished, AI-written TL;DR summary rather than the heuristic sentence-extraction version.

What does the output file look like?＋

Markdown files in data/sn_outputs/. One .md file per episode with clear sections for TL;DR, takeaways, timestamps, resources, and SEO meta. Paste directly into your podcast host's show notes field (Buzzsprout, Podbean, Anchor, etc.) or drop it into your CMS.
